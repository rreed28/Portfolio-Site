In February of 2019 I worked on a freelance redesign project via UpWork for Hayride Scandal. I started with a WordPress theme and was provided a file with the new colors, homepage, and contact form page.

  • The process started with me implementing the new fonts and colors throughout the WordPress site within the admin and CSS styles via FTP
  • Next, I started with each module of the front page going in the WordPress files via FTP and the WordPress Admin
  • Lastly, I created the new contact form with a WordPress plugin and implemented some changes in the admin

Throughout the process we ended with the result live on their site now. If you click here you will also see the document provided of design.
