In my Technology and Learning class we were taught how to create instructional teaching methods through different tools. We had multiple projects throughout the semester where we would have to discover a new technology, learn it, and then create a tutorial to teach others how to use the new technology that we discovered. The tutorials I created during the class were for Studyblue.com and Weebly.com. Some skills I learned in this class were how to create a lesson plan, how to create tutorials, how to discover new technologies/master them, and ways to create online tutorials.
